What I do

HTML

Using HTML to organize, format, and display a web page's content. With tags, elements and attributes I can define the headings, paragraphs, images and other content of a web page. I am able to create pages that allow you to navigate sections within websites and use external links.

CSS

I am able to use Cascading Style Sheets to add style to a web page by dictating how a site is displayed on a browser. I am able to add an external style sheet with classes and ids that allow me to control the text style, size, positioning, color, and more on a website.

JavaScript

I use JavaScript to create all of my applications. This gives me the ability to provide users with dynamic content on each web page or app. I am able to create functions and add interactive items so the user can navigate each web page and operate each application.
